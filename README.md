# 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resume quality using ATS-inspired checks and provides smart suggestions for improvement.

## 🚀 Features

- 📄 Upload Resume
- 📊 ATS Score
- ✅ Resume Analysis
- 💡 Smart Suggestions
- 📱 Responsive Design
- 🌙 Modern UI
- ⚡ Fast & Lightweight
- 🔑 No API Required

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
